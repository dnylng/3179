World Happiness Report 2023

On the left is the average happiness level of each region and subregions inside of a boxplot. We can see that Europe and Oceania have the highest average happiness level, has the lowest average happiness level.

The top boxplot represents the happiness levels of each major region we can see that Africa has on average the lowest Happiness score compared to all the other major regions

The bottom boxplot represents the happiness levels of each minor region

On the left is a scatterplot which compares the happiness level with many other factors which influence it

On the bottom is a pie chart which shows the happiness and its explained by factors, please be aware that all the values in the explained by do not neccessarily add up to the happiness score because the factors just show how much it affects the happiness score, not the actual happiness score itself.

From the data we can see the GDP and life expectancy are the most important factors when it comes to the world happiness level but it is not the do all or end all.

We can see that the countries with the highest happiness level are countries that speak a European Language are the happiest countries. This is consistent with the data in the boxplots as they show that European countries are on average the happiest countries with barely any outliers
